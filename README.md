# Gold Price Movement Prediction (UP/DOWN)

## Overview
This project predicts whether the gold price will increase or decrease based on historical financial indicators using Machine Learning.

## Features
- Data preprocessing and cleaning
- Feature correlation analysis
- Model training
- Prediction: Gold price UP or DOWN

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
